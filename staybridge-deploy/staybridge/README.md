# StayBridge 🏨

A full-featured hotel booking platform built with React + Vite.

## Features
- 🔍 Search hotels with filters and map view
- 📅 Real date picker calendar
- 💳 Payment flow with card entry
- 🎫 Booking confirmation with reference number
- 📊 Hotel owner dashboard with earnings
- 🏷 Deals & offers page
- ❓ FAQ / Help center
- 👤 User accounts with booking history
- ⭐ Write reviews for past stays
- 📱 Mobile responsive layout

## Deploy in 60 seconds

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new)

1. Go to [vercel.com](https://vercel.com) and sign up free
2. Click "Add New Project"
3. Upload this folder or connect GitHub
4. Click Deploy — you get a live URL instantly

## Run locally

```bash
npm install
npm run dev
```

## Buy a domain
After deploying, buy a domain at [namecheap.com](https://namecheap.com) (~$13/year)
and add it in Vercel → Settings → Domains.

## Monetization
- Booking commission via Stripe Connect (12% per booking)
- Hotel subscription tiers ($49/$149/month)
- Featured listing slots ($29-99/month)
